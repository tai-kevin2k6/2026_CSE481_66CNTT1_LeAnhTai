# 2026_CSE481_66CNTT1_LeAnhTai

## Mô tả dự án
Dự án này là bài tập lớn của môn Công Nghệ Phần Mềm - Đại học Thủy Lợi
Môn học này chủ yếu là kiến thức về làm việc nhóm

## Cấu trúc dự án

### Backend (C# - Phát triển thủ công)
- **Công nghệ**: C# (.NET/ASP.NET Core)
- **Mô tả công việc**: 
  - Xây dựng API và logic xử lý dữ liệu
  - Quản lý cơ sở dữ liệu
  - Xác thực và phân quyền người dùng
  - Xử lý business logic
  
**Các tệp backend chính:**
- `/Controllers` - Các endpoint API
- `/Services` - Logic nghiệp vụ
- `/Models` - Mô hình dữ liệu
- `/Database` - Cấu hình cơ sở dữ liệu

### Frontend (HTML/CSS - Phát triển với AI)
- **Công nghệ**: HTML, CSS, JavaScript
- **Mô tả công việc**: 
  - Tạo giao diện người dùng
  - Tích hợp với API backend
  - Responsive design
  - Tương tác người dùng
  
**Các tệp frontend chính:**
- `/Public` - HTML pages
- `/Styles` - CSS files
- `/Scripts` - JavaScript files
